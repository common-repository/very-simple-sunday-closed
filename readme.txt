== DESCRIPTION ==
= About =
This is a very simple plugin to close your website on sundays.

Every sunday your site is closed and a custom page is displayed.

= NOTICE = 
This plugin will be removed from repository. 

There will be a replacement soon: Very Simple Website Closed

= How to use =
After installation go to Settings > Sunday Closed.

Here you can activate and customize the closed on sundays page. 

A default notification will be displayed if title and content are not set.

Default colors are white (background) and dark grey (text).

You can also preview the closed on sundays page.

== Frequently Asked Questions ==
= Why close a website on sundays? =
There are people who believe sunday is a day of worship and rest. Being online is not part of this.

= How can I set my local timezone? =
It uses the timezone set in Settings > General.

= How can I change font? =
Plugin uses the font-family of your theme.

= Can I preview the closed on sundays page? =
Yes, plugin has a preview mode.
